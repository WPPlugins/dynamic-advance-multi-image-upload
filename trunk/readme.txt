== Description ==

DAMIU is a very simple image upload plugin. Any user can upload multiple image from any page or post. Also they can put image title & description.

Features:

* Settings for page or post.
* Settings for title & description.
* Can use any individual page or post.


== Installation ==

Steps

1. Download the plugin & Upload to your '/wp-content/plugins/' directory.
2. Activate the plugin through the 'Plugins' menu in WordPress
3. You can see the "DAMIU Settings" menu in Admin Panel.
4. Copy the shortcode e.g [dmiu dmiu_id=1] from your selected page & paste in your post/page or in your php template.
